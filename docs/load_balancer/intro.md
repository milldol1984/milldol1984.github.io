Cân bằng tải (Cloud Load Balancer) là giải pháp cân bằng tải trên nền điện toán đám mây. 
Cân bằng tải giúp bạn chia tải giữa các Cloud Server (và máy chủ của riêng bạn) nhằm tăng tốc độ xử lý ứng dụng.
